vports - Porting software to Void Linux


Download:
Current vports tree: https://github.com/glowiak/vports/releases/download/vports-tree-current/vports-20202411.tgz


Installation:
# cd /
# tar xzvf /path/to/vports/tree.tgz

Installing packages:
# cd /usr/vports/<category>/<package>
# bash install

Removing packages:
# cd /usr/vports/<category>/<package>
# bash deinstall
